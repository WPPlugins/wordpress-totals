Display your sites total comments, users, and posts all in one widget. 

== Description ==

This will add a widget to your site that will show visitors your blog stats on how many users, posts, and comments your site has. You can configure the heading text and the before and after text of the widget.

Want Total Control?

If you want more control over showing your stats. We also have all three stats in separate plugins.

Total Users: http://vasthtml.com/js/total-users-plugin/

Total Comments: http://vasthtml.com/js/total-comments-plugin/

Total Posts: http://vasthtml.com/js/total-posts-plugin/

Each plugin has a php tag you can insert into your template files.  

== Installation ==

This section describes how to install the plugin and get it working.
 
1. Upload `wordpress-totals` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
